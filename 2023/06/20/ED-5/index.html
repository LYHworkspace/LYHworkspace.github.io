ED-5 | 四维形式电动力学

发表于 2023-06-20 分类于 Physics ， Electrodynamics

Abstract
本篇主要内容为狭义相对论和四维形式电动力学。

闵氏空间与四维语言

自由粒子的拉氏量与运动方程

在闵氏空间中，我们希望力学系统的作用量是一个 Lorentz 标量。对于自由粒子而言，作用量只能取世界线微元 \(\mathrm{d}s\) 的积分

\[ S = -\alpha \int \mathrm{~d}s \]

其中，\(\alpha\) 是待定常数，为了遵循最小原理，前面加一个负号以保证 \(\alpha\) 是正数。这是因为欧式空间中的测地线是取极小值，而闵氏空间中的测地线是取极大值。\(\mathrm{~d}s\) 是闵氏空间中的不变间隔

\[ \mathrm{~d}^2s = \mathrm{d}x^\mu\mathrm{d}x_\mu \]

接下来确定 \(\alpha\) 的值，为了能使 \(c\to\infty\) 极限下退回到经典的情况，注意到

\[ \mathrm{~d}s = \sqrt{c^2\mathrm{d}t^2-\boldsymbol{x}^2}=c\sqrt{1-\frac{\boldsymbol{v}^2}{c^2}} \mathrm{~d}t \]

因此作用量可以改写为

\[ S = - \alpha \int \mathrm{~d}s=\int - \alpha c\sqrt{1-\frac{\boldsymbol{v}^2}{c^2}} \mathrm{~d}t \]

因此我们可以写出“拉氏量”

\[ L= - \alpha c\sqrt{1-\frac{\boldsymbol{v}^2}{c^2}} \approx -\alpha c + \frac{\alpha \boldsymbol{v}^2}{2c} \xrightarrow{c\to\infty} \frac{1}{2} m \boldsymbol{v}^2 \]

前面的常数项 \(-\alpha c\) 在拉氏量中不重要，因此取 \(\alpha = m\)。由此我们得到了经典拉氏量的形式

\[ L=-mc\sqrt{1-\frac{\boldsymbol{v}^2}{c^2}}=- \frac{m c}{\gamma} \]

但是这个形式的拉氏量并不是 Lorentz 标量，因为在把作用量改为对时间的积分时，已经默认了选择某个惯性系在其中展开，这已经失去了普遍性。所以严格来说，四维时空中并没有经典意义上的拉氏量。

闵氏空间中的拉氏量
前面提到，在闵氏空间中经典拉氏量 \(L\) 不是一个 Lorentz 不变量。
在研究单粒子时，可以按如下方式定义相对论性拉氏量 \[ S = \int \mathbb{L} \mathrm{~d}s \] 其中 \(\mathbb{L}=-mc\) 是一个 Lorentz 标量。
在场论中一般使用拉氏量密度 \[ S =\int L \mathrm{~d}t =\int \mathrm{d}^3x~\mathcal{L} \mathrm{~d}t = \int \mathcal{L} \mathrm{~d}^4x \] 该式中作用量 \(S\)，四维时空测度 \(\mathrm{d}^4 x\)，拉氏量密度 \(\mathcal{L}\) 都是 Lorentz 不变量。

我们说，四维时空中并没有经典意义上的拉氏量，是指没有一个拉氏量作为 Lorentz 标量对时间积分后的作用量也是 Lorentz 标量，这并不意味着经典拉氏量不能用。使用相对论性拉氏量 \(\mathbb{L}\) 时，不能直接代入三维形式的欧拉-拉格朗日方程，而是需要使用变分法导出运动方程；如果写出了经典拉氏量 \(L\)，可以直接代入三维形式的欧拉-拉格朗日方程得到运动方程。

三维形式推导

根据经典拉氏量

\[ L=-mc\sqrt{1-\frac{\boldsymbol{v}^2}{c^2}} \]

可得动量

\[ \boldsymbol{p} = \frac{\partial L}{\partial \boldsymbol{v}} = \frac{m\boldsymbol{v}}{\sqrt{1-\frac{\boldsymbol{v}^2}{c^2}}} = \gamma m \boldsymbol{v} \]

哈密顿量

\[ H= \boldsymbol{p} \cdot \boldsymbol{v} - L = \frac{mc^2}{\sqrt{1-\frac{\boldsymbol{v}^2}{c^2}}} = \gamma mc^2 \]

代入欧拉-拉格朗日方程

\[ \frac{\partial L}{\partial \boldsymbol{x}} - \frac{\mathrm{d}}{\mathrm{d}t} \frac{\partial L}{\partial \boldsymbol{v}} = 0 \quad \Longrightarrow \quad \frac{\mathrm{d}}{\mathrm{d}t} (\gamma \boldsymbol{v}) = 0 \]

四维形式推导

对作用量 \(S\) 取变分，有

\[ \delta S = -mc \int \delta \sqrt{\mathrm{d}x^\mu\mathrm{d}x_\mu} = -mc \int \frac{\mathrm{d}x_\mu \delta \mathrm{d}x^\mu}{\mathrm{d}s} \]

定义协变四矢量

\[ u_\mu= \frac{\mathrm{d}x_\mu }{\mathrm{d}s} \]

为粒子的四维速度，则

\[ \delta S = -mc \int u_\mu \delta \mathrm{d}x^\mu = -mc \int u_\mu \mathrm{d}(\delta x^\mu) = -mcu_\mu \delta x^\mu + mc \int \frac{\mathrm{d}u_\mu}{\mathrm{d}s} \delta x^\mu \mathrm{d}s \]

对于端点固定的世界线，上下限处有 \(\left.\delta x^\mu \right|_{\text{ep}} = 0\)，由 \(\delta S=0\) 得自由粒子的运动方程

\[ \frac{\mathrm{d}u_\mu}{\mathrm{d}s} = 0 \quad \Longleftrightarrow \quad \frac{\mathrm{d}^2x_\mu}{\mathrm{d}s^2} = 0 \]

同时可以定义与 \(x^\mu\) 共轭的四动量

\[ p^\mu = mcu^\mu = mc \frac{\mathrm{d}x^\mu}{\mathrm{d}s}=\left( \frac{E}{c}, \;\;\boldsymbol{p} \right) \]

在这个推导过程中，\(\mathbb{L}\) 毫无存在感，这是由于自由粒子比较特殊。以粒子受到一个标量场的作用为例，作用量为 \[ S = -mc \int e^{\Phi(x)} \mathrm{~d} s \] 此时拉氏量 \(\mathbb{L} = -mce^{\Phi(x)}\) 就会参与到积分中
